# Prescripto | AI Hackathon Cluj 2026

**Prescripto** is an AI-driven solution developed to digitize handwritten medical prescriptions and improve patient adherence. This project was built during the **AI Hackathon Cluj 2026**.

## 🚀 Key Features
* **Handwriting Recognition (OCR):** Transcribes medical prescriptions using Computer Vision.
* **AI Interpretation:** Uses an LLM to translate complex medical jargon into natural, easy-to-understand language.
* **Medication Insights:** Extracts dosages, frequency, and active ingredients (DCI) from a pharmaceutical database.
* **Smart Scheduling:** Automatically synchronizes the treatment plan with **Google Calendar** via API for real-time tracking.

## 🛠️ Tech Stack
* **Language:** Python
* **AI/ML:** OCR , LLM Integration
* **APIs:** Google Calendar API
